# CS50’s Introduction to Artificial Intelligence with Python

This repository contains my personal solutions and experiments for **CS50’s Introduction to Artificial Intelligence with Python (CS50AI)**. It is meant to document my learning journey in AI, not to provide reference solutions for others.

> **Important:** If you are currently taking CS50AI (or any similar course), you must not copy, adapt, or submit any part of this code as your own work. Doing so may violate academic honesty policies at both CS50 and your institution [web:11].

---

## Repository structure

- `degrees/` – Implementations for a project involving graph search on a movie/actor dataset (for example, finding degrees of separation between people).
- `tictactoe/` – Implementations of a Tic-Tac-Toe AI using search techniques (for example, minimax).
- Additional folders may appear over time as I progress through the course and add more projects.

The exact project specifications come from CS50AI’s problem sets and are not reproduced here; please refer to the course website for official descriptions [web:11].

## Environment and setup

This repository targets Python as used in CS50AI.

Basic steps to run a project:

1. Create and activate a virtual environment.
2. Install dependencies:

   ```bash
   pip install -r requirements.txt
   ```

3. Navigate into the desired project folder (for example, `degrees/` or `tictactoe/`) and follow the instructions in that directory or in the original CS50AI problem specification [web:11].
